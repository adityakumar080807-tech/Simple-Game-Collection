# 🎮 Simple Game Collection (Python)

This project is a simple Python-based game collection that allows users to play two fun games: Stone-Paper-Scissors and Dice Roll. It is designed as a menu-driven application to provide an interactive and user-friendly experience.

## 🚀 Features
- Play Stone-Paper-Scissors against the computer
- Play Dice Roll Game with random outcomes
- Menu-driven system using infinite loop (`while True`)
- Random result generation using Python's `random` module
- Clear display of winner after each round

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Clone the repository
2. Open the project in VS Code or any Python IDE
3. Run the file:

   ```bash
   python game.py
